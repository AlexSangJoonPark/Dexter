Checker Code
  BX_UNBOXING_IMMEDIATELY_REBOXED
Description
  Boxed value is unboxed and then immediately reboxed
Severity
Major
Detector / Bug Pattern
FindPuzzlers / Bx
More Information

A boxed value is unboxed and then immediately reboxed. For example, in below example problem is that we're converting Long -> long -> Long as Employee limit field is of type Long So in the background:
1. Long.valueOf(lmt) converts Long to long
2. emp.setLimit(); converts long to Long again

Bad Code 
Employee emp = new Employee()
Long lmt = 123L; // The long value is boxed Here

emp.setLimit(Long.valueOf(lmt)); //Then Unboxed and Reboxed at this line
Good Code 
Employee emp = new Employee()
long lmt = 123L;

emp.setLimit(lmt); //Correct Code
