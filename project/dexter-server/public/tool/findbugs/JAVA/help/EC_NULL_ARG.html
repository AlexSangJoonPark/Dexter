Checker Code
  EC_NULL_ARG
Description
  Call to equals(null)
Severity
Critical
Detector / Bug Pattern
FindRefComparison / EC
More Information

This method calls equals(Object), passing a null value as the argument. According to the contract of the equals() method, this call should always return false.

Bad Code 
void compareWithNull(ArrayList paramLst)
{
    ArrayList keyLst = null;
    paramLst.equals(keyLst) // Incorrect - It will always return false
}
Good Code 
void compareWithNull(ArrayList paramLst)
{
    ArrayList keyLst = null; // (Either) Initialize the List if u are creating it

    if(keyLst!= null){ // (Or) Have a null check on it if getting it as input from other code
        paramLst.equals(keyLst) 
    }
}
