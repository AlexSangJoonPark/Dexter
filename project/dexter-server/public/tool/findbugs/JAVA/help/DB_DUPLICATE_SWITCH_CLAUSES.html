Checker Code
  DB_DUPLICATE_SWITCH_CLAUSES
Description
  Method uses the same code for two switch clauses
Severity
Major
Detector / Bug Pattern
DuplicateBranches / DB
More Information

This method uses the same code to implement two clauses of a switch statement. This could be a case of duplicate code, but it might also indicate a coding mistake.

Bad Code 
switch (someIntegerValue) {
   case 1:
      DoSomethingForCase1();
      break;
   case 2:
      DoSomethingForCase1(); // Incorrect : It looks a mistake of duplicated code
      break;
   case 3:
      DoSomethingForCase3();
      break;
}
Good Code 
switch (someIntegerValue) {
   case 1: //FALL-THROUGH
   case 2:		    // Correct : It clearly indicates Intentionally done
      DoSomethingForCase1();
      break;
   case 3:
      DoSomethingForCase3();
      break;
}
