Checker Code
  EC_UNRELATED_TYPES_USING_POINTER_EQUALITY
Description
  Using pointer equality to compare different types
Severity
Critical
Detector / Bug Pattern
FindRefComparison / EC
More Information

This method uses using pointer equality to compare two references that seem to be of different types. The result of this comparison will always be false at runtime.

Bad Code 

void checkEquals(String str, List message)
{
    if(str == message) // Always false at runtime because uncompatible type
}
Good Code 
void checkEquals(String str, List message,String str1)
{
    if(str == str1) // Only Correct to compare two similar types
}
