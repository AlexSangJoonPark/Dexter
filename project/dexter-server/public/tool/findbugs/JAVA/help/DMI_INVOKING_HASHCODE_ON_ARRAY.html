  DMI_INVOKING_HASHCODE_ON_ARRAY
Description
  Invocation of hashCode on an array
Severity
Major
Detector / Bug Pattern
FindPuzzlers / DMI
More Information

The code invokes hashCode on an array. Calling hashCode on an array returns the same value as System.identityHashCode, and ingores the contents and length of the array. If you need a hashCode that depends on the contents of an array a, use java.util.Arrays.hashCode(a). For any two arrays a and b such that Arrays.equals(a, b), it is also the case that Arrays.hashCode(a) == Arrays.hashCode(b).

void checkHashcode(String [] bankNames,String [] passportNames){
	
	if(bankNames.hashcode()==passportNames.hashcode()} // Incorrect : Always FALSE as both hashcode be different unless point to same object.

}
Good Code 
void checkHashcode(String [] bankNames,String [] passportNames){
	
	if(Arrays.hashCode(bankNames) == Arrays.hashCode(passportNames)} // Correct : Returns a hash code based on the contents of the specified array.
	{
		System.out.println("Hashcode Equals"); 
	}
}
