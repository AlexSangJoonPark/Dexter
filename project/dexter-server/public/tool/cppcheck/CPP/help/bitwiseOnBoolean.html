  bitwiseOnBoolean
Description
  Avoid boolean variable in bitwise operation.
Severity
Major
Detector / Bug Pattern
Avoid to use boolean variable in bitwise operation.
More Information
|| and && are boolean operators and are guaranteed to return either 1 or 0. Nothing else. |, & and ^ are bitwise operators. When the domain of numbers you operate on is just 1 and 0, then they are exactly the same, but in cases where your booleans are not strictly 1 and 0--as is the case with the c language--you may end up with some behavior you didn't want. For instance:
Bad Code 
struct Bool
{
	bool    value;	
	void operator&=( bool const v ) { value = value && v; }
	operator bool() const { return value; }
};

int main()
{
	using namespace std;	
	Bool a  = {true};
	a &= true || false;
	a &= 1234;
	cout << boolalpha << a << endl;
	
	bool b = {true};
	b &= true || false;
	b &= 1234;
	cout << boolalpha << b << endl;
}
/*
Output with Visual C++ 11.0 and g++ 4.7.1:
true
false
*/
