Good Code 
/*
Negative integers on right-hand side is undefined behavior in the C language.
The integer promotions are performed on each of the operands. The type of the result 
is that of the promoted left operand. If the value of the right operand is negative or
is greater than or equal to the width of the promoted left operand, the behavior is undefined.
*/
