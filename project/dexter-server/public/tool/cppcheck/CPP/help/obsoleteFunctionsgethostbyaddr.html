Description

Severity
Minor
Detector / Bug Pattern
Avoid usage of the obsolete function 'gethostbyaddr'.
More Information
The function 'gethostbyaddr' is obsolete. Applications should use getaddrinfo(3) and getnameinfo(3) instead.
