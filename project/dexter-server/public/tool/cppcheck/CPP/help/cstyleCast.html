  cstyleCast
Description
  C-style pointer casting.
Severity
Minor
Detector / Bug Pattern
C-style pointer casting.
More Information
C-style pointer casting.
Bad Code 
MyClass *m = (MyClass *)ptr; //CStyle cast
MyClass *m = (MyClass *)ptr; //CStyle cast

char c = 10;  // 1 byte
int *p = (int*)&c; 
/* 4 bytesSince this results in a 4-byte pointer pointing to 1 byte of allocated
memory, writing to this pointer will either cause a run-time error or will overwrite 
some adjacent memory.*/
*p = 5;
/* run-time error: stack corruptionIn contrast to the C-style cast, the static cast will allow
 the compiler to check that the pointer and pointer data types are compatible, which
 allows the programmer to catch this incorrect pointer assignment during compilation.*/
Good Code 
MyClass *m = static_cast(ptr);
MyClass *m = dynamic_cast(ptr);
char c = 10;  // 1 byte
int *q = static_cast(&c); // compile-time errorReinterpret cast
