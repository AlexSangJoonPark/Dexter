Checker Code
  negativeIndex
Description
  Negative array index.
Severity
Critical
Detector / Bug Pattern
Negative array index.
More Information
Array index should not be negative.
Bad Code 
int arr[10];
int x = arr[-2]; // invalid; out of range	

int a[10] = { /* whatever */ };
int i;
b = a + 10; /* one past end */
	
for (i = -10; i < 0; i++)
{
	int x = b[i]; /* all valid */
}	
....but b[-11] is invalid.
Good Code 
int arr[10];
int* p = &arr[2];
int x = p[-2]; 
// valid:  accesses arr[0]
