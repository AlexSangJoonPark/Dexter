Detector / Bug Pattern
size of void.
More Information
Behaviour of sizeof(void) is not covered by the ISO C standard. A value for sizeof(void) is defined only as part of a GNU C extension, which defines sizeof(void) to be 1
Bad Code 
int main (void)
{
	//The type void has no size; that would be a compilation error. 
	void n;	
	void * p = (void*) malloc(sizeof(void));	
	//Depend on the compiler it will return the size of void pointer.	
}
Good Code 
int main(void)
{
	int num = 0;
	void *ptr;
	num = 15;
	ptr = &num;
	/* Output the size   */
	printf("Size of Pointer : %d Bytes",sizeof(ptr)); //It will print 2 byte.
	return 0;
}
