Description
  Redundant c_str()-conversion in parameter passing.
Severity
Major
Detector / Bug Pattern
Redundant c_str()-conversion in parameter passing.
More Information
Passing the result of c_str() to a function that takes std::string as argument 0 is slow and redundant
Bad Code 
void myFunc(std::string str)  
{
    char *buf = str.c_str;    
    std::cout << str;
}
int main()
{
    const std::string str = "hello world";
    myFunc(str);
    return 0;
}	
LegacyFunction(ModernFunction().c_str());
Good Code 
void myFunc(std::string str)  
{    
    const char* buf = str.c_str();
    std::cout << str;
}
int main()
{
    const std::string str = "hello world";
    myFunc(str);
    return 0;
}
