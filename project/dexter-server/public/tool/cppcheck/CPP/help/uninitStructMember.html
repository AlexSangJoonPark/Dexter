  uninitStructMember
Description
  Uninitialized struct member.
Severity
Major
Detector / Bug Pattern
Uninitialized struct member.
More Information
Uninitialized struct member.
Bad Code 
Example 1:					
struct T
{
    int x;
    int y;
};
struct T t; // t.x, t.y will NOT be initialized to 0 (not guaranteed to)

Example 2:
int x[10]; // Won't be initialized.
Good Code 
Example 1:					
struct T
{
    int x;
    int y;
};
struct T t = {42}; // t.y will be initialized to 0.	

Example 2:	
int x[10] = {1}; // initialized to {1,0,0,...}
