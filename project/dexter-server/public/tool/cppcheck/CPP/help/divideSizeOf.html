/*			
Note: 
On most machines and ABIs, pointers are generally word-aligned. So dividing by the sizeof pointers is 
essentially ignoring the least bits (e.g. 3 lowest bits of bytes addresses are 0 on most 64 bits processors
since a 64 bits word has 8 bytes, each of 8 bits).*/
