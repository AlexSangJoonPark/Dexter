Checker Code
  BX_BOXING_IMMEDIATELY_UNBOXED_TO_PERFORM_COERCION
Description
  Primitive value is boxed then unboxed to perform primitive coercion
Severity
Major
Detector / Bug Pattern
FindPuzzlers / Bx
More Information

A primitive boxed value constructed and then immediately converted into a different primitive type (e.g., new Double(d).intValue()) . Rather, Just perform direct primitive coercion (e.g., (int) d).

Bad Code 
new Double(d).intValue() // Incorrect
Good Code 
(int) d // Correct
