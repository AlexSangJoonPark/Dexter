Description
  Covariant compareTo() method defined
Severity
Critical
Detector / Bug Pattern
FindHEmismatch / Co
More Information

This class defines a covariant version of compareTo().  To correctly override the compareTo() method in the Comparable interface, the parameter of compareTo() must have type java.lang.Object.

Bad Code 
 class Motor implements Comparable {
    private int a;
    private int b;

    public int compareTo(Motor motor) { // Incorrect : parameter is of type Motor
        // compare using a and b
    }
}
Good Code 
class Motor implements Comparable {
    private int a;
    private int b;

    public int compareTo(Object motor) { // Correct : parameter is of type java.lang.Object
		if (motor instanceof Motor)
		{
			// compare using a and b
		}
		...
	}
}
