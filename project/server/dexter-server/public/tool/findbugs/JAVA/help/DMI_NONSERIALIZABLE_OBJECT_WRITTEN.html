Checker Code
  DMI_NONSERIALIZABLE_OBJECT_WRITTEN
Description
  Non serializable object written to ObjectOutput
Severity
Critical
Detector / Bug Pattern
FindUseOfNonSerializableValue / DMI
More Information

This code seems to be passing a non-serializable object to the ObjectOutput.writeObject method. If the object is, indeed, non-serializable, an error will result.

Bad Code 
FileOutputStream fos = new FileOutputStream("t.tmp");
ObjectOutputStream oos = new ObjectOutputStream(fos);
Person person  = new Person(); // Incorrect: Will throw NotSerializableException, If Class Person is non-serializable
oos.writeObject(person);

oos.close();
