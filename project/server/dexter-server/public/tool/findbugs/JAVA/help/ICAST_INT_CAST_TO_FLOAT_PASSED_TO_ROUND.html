Checker Code
  ICAST_INT_CAST_TO_FLOAT_PASSED_TO_ROUND
Description
  int value cast to float and then passed to Math.round
Severity
Critical
Detector / Bug Pattern
IDivResultCastToDouble / ICAST
More Information

This code converts an int value to a float precision floating point number and then passing the result to the Math.round() function, which returns the int/long closest to the argument. This operation should always be a no-op, since the converting an integer to a float should give a number with no fractional part. It is likely that the operation that generated the value to be passed to Math.round was intended to be performed using floating point arithmetic.

Bad Code 
// The code ((size * 10) / 1024) is a whole number and if you round a whole you should get the same number every time
string data = (Math.round(((size * 10) / 1024)) / 10) + " KB";
Good Code 
// Perhaps what you intended was to round down
string data = size * 10 / 1024 / 10.0 + " KB";

//or almost the same thing using Math.round
string data = return Math.round(size / 102.4) / 10.0 + " KB";
// This will give you the number KB with one decimal place.
