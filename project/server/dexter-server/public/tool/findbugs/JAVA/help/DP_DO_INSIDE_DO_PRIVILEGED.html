Checker Code
  DP_DO_INSIDE_DO_PRIVILEGED
Description
  Method invoked that should be only be invoked inside a doPrivileged block
Severity
Critical
Detector / Bug Pattern
DoInsideDoPrivileged / DP
More Information

This code invokes a method that requires a security permission check. If this code will be granted security permissions, but might be invoked by code that does not have security permissions, then the invocation needs to occur inside a doPrivileged block.

Bad Code 
System.loadLibrary("awt"); // Incorrect Invocation with out doPriviledged() Block (Example 1)
String user = System.getProperty("user.name")  // Incorrect Invocation  with out doPriviledged() Block (Example 2)
Class myObjectClass = classLoader.loadClass("reflection.MyObject"); // Incorrect Invocation with out doPriviledged() Block(Example 3)
Good Code 
somemethod() {
       ...normal code here...
       AccessController.doPrivileged(new PrivilegedAction() {
           public Object run() {
               // privileged code goes here, for example:
               System.loadLibrary("awt");
               return null; // nothing to return
           }
       });
       ...normal code here...
  }
