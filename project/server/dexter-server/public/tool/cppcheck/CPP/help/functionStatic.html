More Information
/* The member function 'funcname' can be static. No, a static method can only see static members of the class. It doesn't make sense for it to access instance members (as in, standard variables etc) as they don't exist unless the class has been instantiated. It seems like you want something like the Singleton design pattern. This allows for only a single instance of the class to exist at a time. */
